## HAIN World

**HAIN World** is the public-facing identity of the **Hybrid AI Native** framework.

HAIN is a framework for understanding how modern intelligent systems are evolving across **cloud**, **edge**, and **agentic** contexts through clearer public structure, cleaner architectural language, and structured narrative assets.

## Structure

- **HAIN World** — public entry point
- **HAINOA** — architecture core
- **HAINIZE** — toolkit and practical-entry direction
